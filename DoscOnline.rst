Запрос баланса 
==============

Точка подключения:
   
.. code-block:: json

   https://devino.online
   
   
   
.. code-block:: json

   /api/v1/companies/current_balance
   
 Пример овтета:

.. code-block:: json

   {
    "balance": 0,
    "balance_currency": "string"
   }
   
 
Параметры ответа:
 
+----------------------+------------+--------------------------------------------------------+--------------+
|      Параметр        | Тип данных |    Описание                                            |Обязательный  |
+======================+============+========================================================+==============+
| balance              |   integer  |  Баланс компании                                       |       Да     |
+----------------------+------------+--------------------------------------------------------+--------------+
| balance_currency     |   string   |  Валюта                                                |       Да     |
+----------------------+------------+--------------------------------------------------------+--------------+
